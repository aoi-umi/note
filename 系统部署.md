# docker

- 非root用户使用docker出现权限问题
```bash
sudo groupadd docker #添加docker用户组
sudo gpasswd -a $USER docker #将当前用户添加至docker用户组
newgrp docker #切换docker用户组 或登出用户
```
## 查看上一命令返回码
win
>echo %errorlevel%

linux
>echo $?

0为成功
